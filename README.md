# time_series_project
Final project repository for Spring 2026 DS6425 - Time Series Modeling

This project aims to forecast flood risk for the Charlottesville/Albemarle Co./Fluvanna Co. region. Full data, and code to reproduce EDA, feature engineering, and modeling is included. 

* `final_project_eda.ipynb` – exploratory data analysis and figures
* `final_project_features.ipynb` – feature engineering
* `final_project_modeling_v1.ipynb` – modeling and figures
